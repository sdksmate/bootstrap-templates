Cartzilla is the ultimate e-commerce front-end template based on the latest Bootstrap 5 framework. Along with styling all the default Bootstrap components it introduces lots of new flexible elements you can use across website. 40+ components in total (plus sign means some components have multiple style variations). Besides well thought out Store Front pages (Homepage, Catalog, Categories), Cartzilla includes pages of User Account, Cart, Checkout, etc. to ensure your online shop will provide smooth and engaging user experience. Every successful e-commerce website should have Blog (News) section, About Us information, Contact details, Help / FAQ section to engage with its visitors. Cartzilla includes all this along with service pages like: 404 Not Found, etc. More Front pages and Secondary pages are coming in the future updates.  

Cartzilla Features

  • 7 Demos: Digital Marketplace Template, Grocery & Supermarket Store, Clothing Store, Electronics Store, Food Delivery Service, Single Product / Mono Brand Store
  • Single / Multi-vendor Store
  • 40+ Flexible UI Components
  • 9+ User Account Pages
  • 7 Vendor Dashboard Pages
  • Shop catalog Grid / List layout options + Shop Categories
  • 3 Product Page Template variations
  • Complete Order Workflow: Cart + Checkout
  • Secondary pages: About, Contacts, Help Center, 404, etc.
  • Blog page templates with different layouts
  • 9 Header (Navbar) variations
  • Built with latest Bootstrap – 100% Responsive
  • No jQuery dependecy. All plugins and custom (core) code is written in Vanilla JS using ES6 moules.
  • Toolbar with Cart, Menu, Filters, etc. buttons designed specifically for handheld devices to improve user experience on the go.
  • Kick-start your development with Node.js / Gulp setup
  • Speed up customization with Sass and Pug (aka Jade) preprocessors
  • Use Pug – Node Templating Engine or vanilla HTML
  • Mobile Friendly Interface + Touch-enabled Sliders
  • Interactive Credit Card Checkout Payment step
  • Mega-menu functionality
